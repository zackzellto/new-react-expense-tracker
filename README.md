# React Expense Tracker

React Expense Tracker is an application that allows you to input expenses with details about each.

[Heroku Link](https://zack-react-expense-tracker.herokuapp.com/)

## How It's Made

React Expense Tracker is made using React!

## UI
![React Expense Tracker UI](https://github.com/zackzellto/new-react-expense-tracker/blob/main/Expense-Tracker-UI.png?raw=true)

## Usage

1. Input the necessary expense information into the input fields and click "Submit"
2. If an expense is no longer needed or was entered by mistake, you can select "Delete" on the respective row
